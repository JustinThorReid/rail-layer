# MovementBonusEquipmentPrototype
*typename: `movement-bonus-equipment`*
*extends: `EquipmentPrototype`*

Used by [exoskeleton](https://wiki.factorio.com/Exoskeleton). Increases max speed of characters or
acceleration of vehicles if they have this equipment in their grid.

## Properties

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `energy_consumption` | `Energy` | no |  |
| `movement_bonus` | `double` | no | Multiplier of the character speed/vehicle acceleration. |
