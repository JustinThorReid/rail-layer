# EntityGhostPrototype
*typename: `entity-ghost`*
*extends: `EntityPrototype`*

The entity used for ghosts of entities. In-game, the inner entity (the entity this is a ghost of) is
rendered with a [UtilityConstants::ghost_tint](prototype:UtilityConstants::ghost_tint).
