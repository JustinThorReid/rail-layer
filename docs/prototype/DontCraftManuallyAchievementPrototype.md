# DontCraftManuallyAchievementPrototype
*typename: `dont-craft-manually-achievement`*
*extends: `AchievementPrototypeWithCondition`*

This prototype is used for receiving an achievement when the player finishes the game without
crafting more than a set amount.

## Properties

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `amount` | `uint32` | no | This will disable the achievement, if the player crafts more than this. |
