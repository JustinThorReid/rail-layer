# CapsulePrototype
*typename: `capsule`*
*extends: `ItemPrototype`*

A capsule, for example a [combat robot capsule](https://wiki.factorio.com/Combat_robot_capsules) or
the [raw fish](https://wiki.factorio.com/Raw_fish).

## Properties

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `capsule_action` | `CapsuleAction` | no |  |
| `radius_color` | `Color` | yes | Color of the range radius that is shown around the player when they hold the capsule. |
