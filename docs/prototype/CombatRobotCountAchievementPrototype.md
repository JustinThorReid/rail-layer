# CombatRobotCountAchievementPrototype
*typename: `combat-robot-count-achievement`*
*extends: `AchievementPrototype`*

This prototype is used for receiving an achievement when the player has a certain robot follower
count.

## Properties

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `count` | `uint32` | yes | This will trigger the achievement, if player's current robot count is over this amount. |
