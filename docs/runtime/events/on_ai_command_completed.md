# Event: on_ai_command_completed

Called when a unit/group completes a command.

## Event Data

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `unit_number` | `uint32` | no | unit_number/group_number of the unit/group which just completed a command. |
| `result` | `defines.behavior_result` | no |  |
| `was_distracted` | `boolean` | no | Was this command generated by a distraction. |
| `name` | `defines.events` | no | Identifier of the event |
| `tick` | `uint32` | no | Tick the event was generated. |
