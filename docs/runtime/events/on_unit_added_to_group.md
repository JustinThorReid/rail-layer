# Event: on_unit_added_to_group

Called when a unit is added to a unit group.

| ---- | ---- | -------- | ----------- |
| `unit` | `LuaEntity` | no |  |
| `group` | `LuaCommandable` | no |  |
| `name` | `defines.events` | no | Identifier of the event |
| `tick` | `uint32` | no | Tick the event was generated. |
