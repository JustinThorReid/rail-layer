# Concept: EquipmentIDAndQualityIDPair

An equipment prototype with optional quality specification.

## Fields

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `name` | `EquipmentID` | no | Equipment prototype. |
| `quality` | `QualityID` | yes | Quality prototype. Normal quality will be used if not specified. |
