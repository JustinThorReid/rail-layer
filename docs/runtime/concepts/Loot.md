# Concept: Loot

## Fields

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `item` | `string` | no | Item prototype name of the result. |
| `probability` | `double` | no | Probability that any loot at all will drop, as a number in range [0, 1]. |
| `count_min` | `double` | no | Minimum amount of loot to drop. |
| `count_max` | `double` | no | Maximum amount of loot to drop. |
