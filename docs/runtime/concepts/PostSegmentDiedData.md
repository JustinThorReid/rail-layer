# Concept: PostSegmentDiedData

Information about an individual segment in a segmented unit when the unit died.

## Fields

| Name | Type | Optional | Description |
| ---- | ---- | -------- | ----------- |
| `position` | `MapPosition` | yes | The segment's position when the unit died, if it had one. |
| `orientation` | `RealOrientation` | yes | The segment's orientation when the unit died, if it had one. |
